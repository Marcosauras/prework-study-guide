HTML

  • The lang="en" tells the browser the language that will be used for the webpage, in this case it is English.
  • The head element contains information about the webpage, it is not visible to the user on the webpage.
  • The body element represents the visible content that the user is able to see on the webpage.
  • The style element defindess the CSS styles associated with a website.
  • The bases element defines the base URL for a webpage.
  • The link element contains a conection to an external resource to the HTML document.
  • The footer element appears at the bottom of the page this is ussually used for author, copyright, sitemap, and navigation.

CSS

  • CSS is used to define the style.

Git

  • Before doing any command in Git bash make sure to put Git in front of it.
  • We make changes in a branch and not directly to the live site to reduce mistakes.
  • A Pull request or "PR" is asking for permission to merge one branch with another.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git commit -m "descriptive message": will commit our staged content as a new comit snapshot.

JavaScript

  • JavaScript is used to add structure to create a website.
